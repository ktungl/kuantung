JADE — 服飾專賣電商平台

完整的電商解決方案，整合會員、金流、社群與即時通訊功能，專注服飾零售體驗。

專案簡介

JADE 為服飾專賣店打造，用戶可以瀏覽分類精準的商品、進行社群互動、並透過完整的會員等級及金流服務完成交易。

Vue.js ASP.NET Core Azure SQL SignalR

專案亮點

  • 團隊領導：帶領6人開發團隊，2個月完成主要 MVP。
  • 全端開發：前後端分離架構，RESTful API 設計與實作。
  • 系統整合：整合綠界金流、Cloudinary 圖床、OpenAI 等服務。
  • 創新功能：社群電商融合、AI 客服、即時通訊。
範例程式碼
// ASP.NET Controller 範例
public class BannerController : Controller {
  public async Task GetActiveBanners(string position) {
    var banners = await _bannerService.GetActiveBannersByPosition(position);
    return Json(banners.OrderBy(b => b.DisplayOrder));
  }
}
