技術架構

開發模式： 獨力開發（個人專案）

專案類型： 桌面應用程式（Windows Forms）

技術架構： C# WinForms + SQL Server

開發目標： 為獨立書店提供完整的營運管理解決方案

專案背景與價值定位

解決的實際問題

目標客群

缺乏網路商店系統的獨立書店

市場痛點

傳統書店缺乏數位化管理工具，難以有效管理庫存、會員和訂單

商業價值

提供小型書店負擔得起的管理系統，協助傳統書店數位轉型

全功能模組開發
