專案背景與目標

針對現有電商平台痛點，開發專注服飾領域的精準定位電商平台：

解決問題
  • 商品分類過於廣泛
  • 缺乏社群互動
  • 賣家開店門檻高
  • 會員忠誠度機制不完善
核心特色
  • 六大風格分類
  • 社群整合
  • 低門檻開店
  • 完整會員等級制度
技術創新： 整合管理、購物、社群與客服功能，建立以使用者體驗為核心的服飾電商平台

[BackgroundJob]
public async Task PublishScheduledPost(int postId)
{
    var post = await _postService.GetByIdAsync(postId);
    if (post != null && post.ScheduledTime <= DateTime.Now)
    {
        await _postService.PublishPostAsync(postId);
    }
}
