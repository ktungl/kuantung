專案概述

JADE 是一個專注於服飾領域的電商平台，整合了購物、社群、會員管理等多重功能。作為專案組長，我帶領6人團隊歷時3個月，從零開始建構了這個包含60+資料表的完整電商生態系統。

專案亮點：這不僅是一個技術實作專案，更是我在專案管理、系統整合和團隊協作方面的重要里程碑。從資料庫設計到前端交互，從第三方服務整合到部署上線，每個環節都體現了現代全端開發的最佳實踐。

專案背景與挑戰

市場痛點分析

在深入研究現有電商平台後，我們發現了幾個關鍵問題：

🎯

分類過於廣泛

傳統電商平台商品種類繁雜，缺乏針對特定領域的深度優化，導致用戶購物體驗不夠精準。

👥

社群互動缺失

多數電商平台僅專注於交易功能，忽略了用戶之間的互動和社群建立，缺乏穿搭分享等社交元素。

🏪

開店門檻高

中小型賣家難以快速進入市場，複雜的開店流程和高昂的平台費用成為主要障礙。

🎁

會員制度單一

缺乏完整的會員等級制度和忠誠度計畫，無法有效提升用戶黏性和回購率。

解決方案：針對這些痛點，JADE 平台採用了六大風格分類、社群整合、低門檻開店和完整會員等級制度，打造以用戶體驗為核心的服飾電商平台。

  • 💻 核心功能開發：主導首頁管理、部落格系統、社群功能等關鍵模組的開發
  • 🔗 系統整合協調：協調前後端開發，整合各模組功能，解決跨模組相依性問題

資料庫設計精華

設計了包含60+資料表的完整電商資料庫，涵蓋會員系統、商品管理、訂單流程等核心業務邏輯：

// ASP.NET Core 控制器範例
[ApiController]
[Route("api/[controller]")]
public class BannerController : ControllerBase
{
    private readonly IBannerService _bannerService;
    
    public BannerController(IBannerService bannerService)
    {
        _bannerService = bannerService;
    }
    
    [HttpGet("{position}")]
    public async Task<IActionResult> GetActiveBanners(string position)
    {
        var banners = await _bannerService.GetActiveBannersByPosition(position);
        return Ok(banners.OrderBy(b => b.DisplayOrder));
    }
    
    [HttpPost]
    public async Task<IActionResult> CreateBanner([FromBody] BannerCreateDto dto)
    {
        var result = await _bannerService.CreateBannerAsync(dto);
        return Ok(result);
    }
}

[BackgroundJob]
public async Task PublishScheduledPost(int postId)
{
    var post = await _postService.GetByIdAsync(postId);
    if (post != null && post.ScheduledTime <= DateTime.Now)
    {
        post.Status = PostStatus.Published;
        post.PublishedAt = DateTime.Now;
        await _postService.UpdateAsync(post);
        
        // 發送通知郵件

    }

團隊管理與協作經驗

挑戰與解決方案

🎯 進度管控挑戰

問題：6人團隊分工複雜，模組間依賴關係複雜

解決：建立每日站會制度，使用 Trello 進行任務追蹤

🔗 模組整合挑戰

問題：不同模組間 API 格式不統一，整合困難

解決：制定統一的 API 設計規範，建立共用 DTO 類別

📝 溝通協調挑戰

問題：團隊成員技術背景不同，溝通成本高

解決：建立技術文件庫，定期進行技術分享會

⏰ 時程壓力挑戰

問題：3個月開發期限緊迫，功能需求複雜

解決：採用敏捷開發，分階段交付，優先開發核心功能

管理心得：成功的專案管理不僅需要技術能力，更需要良好的溝通技巧和團隊協調能力。透過建立明確的開發規範、定期的進度追蹤和有效的問題解決機制，我們成功在時限內完成了這個複雜的電商平台。
